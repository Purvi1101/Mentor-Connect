# MentorConnect

MentorConnect is a full-stack web application developed to connect students with mentors for academic and career guidance.  
The platform provides separate access for mentors and students along with a simple and effective communication system.

---

## Features
- Mentor & Student Registration and Login  
- Role-based access system  
- Separate chat functionality  
- Secure authentication  
- Responsive user interface  
- MySQL database integration  

---

## Tech Stack
**Frontend:** HTML, CSS, JavaScript  
**Backend:** Java (Spring Boot)  
**Database:** MySQL  

---

## Project Overview
This project focuses on building a real-world mentoring platform using Java and Spring Boot.  
It demonstrates full-stack development concepts including authentication, database connectivity, and user interaction.

---

## How to Run
1. Clone the project  
2. Configure MySQL database  
3. Run the Spring Boot application  
4. Open `http://localhost:8080` in browser
